# HTML 5 Details Tag

```html
<details>
  <summary>Details Example</summary>
  <p>Here are the details!</p>
</details>
```

Example:
<details>
  <summary>Click to see more!</summary>
  <p>Reduce screen real estate and hide when not needed.</p>
</details>
