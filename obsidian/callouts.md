# Callouts

Callouts are designated with a chevron `>` followed by a label and optional title.

```md
> [!tip] Title-only callout
```

![Title callout](https://github.com/gkhays/til/assets/773981/ae5a00d7-8e89-42eb-b538-aa9ec65f96e2)

A foldable callout can be create with a plus (+) or minus (-) sign.

```md
> [!info]- Exapnd to see more
> This is the detail area.
```

![Expandable](https://github.com/gkhays/til/assets/773981/c7237170-feeb-432d-8bc4-ae9af5b21b4f)


### Code

```md
> [!example]-
> ```java
> public static void main(String[] args) {
>    System.out.println("Hello world!);
> }
> ```
```

### Expanded

```md
> [!example]+
> ```java
> public static void main(String[] args) {
>    System.out.println("Hello world!);
> }
> ```
```
