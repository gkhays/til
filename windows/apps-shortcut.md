# Applications Virtual Folder Shortcut

Click in the Explorer address bar and type

```command
shell:AppsFolder
```

![Apps Folder](../images/apps-folder.png)

Right mouse-click and drag to the desktop.

![Create Shortcut](../images/create-shortcut.png)
