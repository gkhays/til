# List in Markdown Table Cell

Use embedded HTML to create a list in a table cell.

| Markdown                                              | Output                                              |
| :---------------------------------------------------- | :-------------------------------------------------- |
| `<list><li>One</li><li>Two</li><li>Three</li></list>` | <list><li>One</li><li>Two</li><li>Three</li></list> |
